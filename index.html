Advanced Lane Finding: Write A Software Pipeline To Identify Lane Boundaries In A Video

[ Github Project Repo ]

Computed camera calibration matrix and distortion coefficients given a set of chessboard images. Applied distortion correction to raw images. Use color transforms, gradients, etc., to create a threshold binary image. Applied a perspective transform to rectify binary image ("birds-eye view"). Detect lane pixels (using histogram peaks) and polynomial fit to find the lane boundary. Implemented margin/band search based on curve fits from previous frame. Determined curvature of the lane and vehicle position with respect to center. Warped the detected lane boundaries back onto the original image. Output visual display of lane boundaries and numerical estimation of lane curvature and vehicle position. Make a pipeline to implement the above set of steps on all frames from the video.
